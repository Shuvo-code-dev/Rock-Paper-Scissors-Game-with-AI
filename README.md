## 📝 About

This is a modern, browser-based implementation of the classic Rock-Paper-Scissors game. Challenge an intelligent AI opponent, track your score, and enjoy an interactive gaming experience with stunning visuals and smooth animations.

## ✨ Features

🎯 **Interactive Gameplay**
- Choose between Rock, Paper, or Scissors with intuitive button controls
- Real-time AI opponent with random decision-making
- Instant result display with emoji indicators

📊 **Score Tracking**
- Keep track of your wins and losses
- Real-time score updates
- Persistent gameplay experience

🎨 **Beautiful Design**
- Modern gradient background (vibrant red to white)
- Smooth button hover animations with scale effects
- Responsive button styling with color-changing interactions
- Clean, centered layout with cornsilk card design
- Professional typography and spacing

⚡ **Smooth Interactions**
- Button hover effects with transform scaling
- Color-changing buttons on interaction
- Clear, bold result messages
- Gradient result display box

## 🚀 How to Play

1. **Open** `Rock-Paper-Scissors.html` in your web browser
2. **Choose** your move: Rock 🪨, Paper 📄, or Scissors ✂️
3. **See** what the AI chose instantly
4. **Win** or lose based on classic game rules
5. **Watch** your score update in real-time

## 📋 Game Rules

| Your Choice | AI Choice | Result |
|:---:|:---:|:---:|
| Rock | Scissors | ✅ You Win! |
| Paper | Rock | ✅ You Win! |
| Scissors | Paper | ✅ You Win! |
| Any | Same | 🤝 Tie! |
| Otherwise | - | ❌ AI Wins! |

## 📦 Installation

### Option 1: Direct Browser Access
1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/Rock-Paper-Game.git
   cd Rock-Paper-Game
   ```
2. Open `Rock-Paper-Scissors.html` in your favorite web browser

### Option 2: Using Python Server (Optional)
```bash
python -m http.server 8000
```
Then open `http://localhost:8000` in your browser
